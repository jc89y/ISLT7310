IBM Maximo: Export data to an external database     

To export data from IBM Maximo requires several steps. You must configure an end point, select an object structure, configure a publish channel, configure an external system, and when exporting to an external database, create an IFACETABLE.

Create End point: To configure a database end point in Maximo you must configure the JDBC database driver, the JDBC URL, and a username and password.

Configure Object Structure: To configure a database end point in Maximo you must configure the JDBC database driver, the JDBC URL, and a username and password.

Configure Publish Channel: To configure a database end point in Maximo you must configure the JDBC database driver, the JDBC URL, and a username and password.

Configure External System: To configure a database end point in Maximo you must configure the JDBC database driver, the JDBC URL, and a username and password.

Create IFACETABLE: To configure a database end point in Maximo you must configure the JDBC database driver, the JDBC URL, and a username and password.

Export Data: Press the EXPORT DATA button on the Publish Channel tab inside the External System.
